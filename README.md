# Global Market Watch Dashboard

A real-time financial dashboard built with **Python** and **Streamlit** to track global equities, commodities, and currencies.

## Features
* **Real-time Data:** Live fetching via Yahoo Finance API.
* **Interactive Charts:** Plotly visualizations with dynamic scaling.
* **Portfolio Analytics:** Hypothetical equal-weighted portfolio simulator.
* **Asset Comparison:** Normalized relative performance charts.

## How to Run
1.  Clone the repo:
    ```bash
    git clone [https://github.com/YOUR_USERNAME/market-watch-dashboard.git](https://github.com/YOUR_USERNAME/market-watch-dashboard.git)
    ```
2.  Install requirements:
    ```bash
    pip install -r requirements.txt
    ```
3.  Run the app:
    ```bash
    streamlit run dashboard.py
    ```
